## 问题描述

**Description**

```
给定一根长度为n(n<=10000)的钢条以及一张价格表, 请计算这根钢条能卖出的最大总收益. 价格表表示为(li,pi), 1<=i<=k. 不在价格表中的钢条可卖出价格为0.
```

**Input**

```
第一行输入m(m<=10)表示有M组数据。每组数据第一行输入两个int型整数n和k，分别表示钢条长度以及价格表中不同价格数量. 接下来一行输入k个价格的表示(li,pi), 均为整数, li可能大于n。
```

**Output**

```
输出m行整数，第i行表示第i组数据的最大总收益。
```

**Sample Input**

```
2
27 3
35 41 61 49 73 74
94 2
21 55 88 64 
```

**Sample Output**

```
0
220
```
