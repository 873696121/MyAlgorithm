## 问题描述

**Description**

```
给定一个长度为N的int型数组a[0,1,2,...N-1], 请计算逆序对个数.当i<j且a[i]>a[j], 则称a[i]与a[j]是一对逆序对.
```

**Input**

```
第一行输入M表示包含M组测试数据，每组先输入N (N<=50000), 接着输入N个int型整数.
```

**Output**

```
输出逆序对个数.
```

**Sample Input**

```
2
5 1 5 2 1 3
6 85 16 44 99 66 1
```

**Sample Output**

```
4
9
```
