## 问题描述

**Description**

```
给定一个N(N<=50000)个int型整数的集合以及一个int型整数X, 问集合中是否存在两个元素的和等于X.
```

**Input**

```
第一行输入M表示有M组测试. 每组测试首先输入N和X，接下来输入N个int型整数.
```

**Output**

```
若否存在两个元素的和等于X则输出yes, 否则输出no.
```

**Sample Input**

```
2
8 7 
1 5 11 5 4 3 9 6
8 7 
1 5 11 5 5 3 9 5
```

**Sample Output**

```
yes
no
```
